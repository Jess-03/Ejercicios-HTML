Definiciones

Función
Una función es un conjunto de instrucciones, cuya finalidad es realizar una tarea específica; tiene valores de entrada y de salida.
let
La palabra "let", es una palabra reservada en JavaScript que se utiliza para declarar una variable; los valores de esa variable pueden cambiar.
const
En JavaScript, la palabra reservada "const" se utiliza para declarar una variable. A diferencia de "let", la variable no puede ser declarada más de una vez.

Etiquetas HTML

a
Esta estiqueta se utiliza para crear o enlazar un hipervínculo que nos lleva a un recurso externo o a una ubicación interna de la página web
button
La etiqueta button, es un elemento interactivo activado por el usuario; una vez activado realiza una acción como "enviar" un formulario.
footer
La etiqueta footer, se utiliza para crear una sección denominada como pie de página
